# Requirements : 
- docker
- docker-compose

# Steps to follow : 

#### Clone the repository

```git clone https://gitlab.com/sf3ris/dashboard-react-rtk my-dashboard-app && cd my-dashboard-app```

##### Build and Run

``` docker-compose up -d --build --force-recreate```
